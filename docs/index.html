Architecture

Five phases. One pipeline.

From raw WebSocket data to trading signal in under 15 milliseconds. Every phase runs on a single laptop.

PHASE 1
The Brain
DistilBERT fine-tuned with LoRA on 16.7K financial sentences. FP16 mixed precision fits in 4GB VRAM.
PyTorchPEFT/LoRAONNX
PHASE 2
Nervous System
Async WebSocket pipeline ingests Binance crypto + Alpaca stock data with zero blocking.
asyncioWebSocketBinanceAlpaca
PHASE 3
The Spine
ONNX Runtime runs inference at ~5ms/prediction on GPU or ~55ms on CPU. GIL-free threading.
ONNX RuntimeCUDANumPy
PHASE 4
The Face
FastAPI server pushes real-time signals to a browser dashboard via WebSocket broadcast.
FastAPIReactWebSocket
PHASE 5
The Edge
Rolling Z-score divergence detection catches reversals. Paper trader manages risk autonomously.

Stack

All open source. All free.

🧠
DistilBERT + LoRA
66M parameter model, only 296K trained via Low-Rank Adaptation. Exports to ONNX for 2-3x faster inference.
⚡
asyncio + WebSockets
Non-blocking concurrent streams from Binance and Alpaca. Single thread handles all I/O while inference runs in parallel.
📊
Divergence Detection
Rolling Z-scores detect when sentiment and price disagree. These contrarian signals precede reversals.
💹
Paper Trading Engine
Full position management: stop-loss, take-profit, trailing stops, fixed fractional sizing, max drawdown circuit breaker.
🌐
FastAPI + React
Real-time dashboard with WebSocket broadcast. Dual-panel view for crypto and stocks simultaneously.
📦
Zero Cost Infrastructure
No paid APIs. No cloud bills. Binance public WebSocket + Alpaca free tier. Runs entirely on your laptop.
